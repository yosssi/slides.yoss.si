Hello world!

package main

import "github.com/go-martini/martini"

func main() {
  m := martini.Classic()
  m.Get("/", func() string {
    return "Hello world!"
  })
  m.Run()
}

Basics

  • Routes
  • Handlers

Routes

  • An HTTP method paired with a URL-matching pattern 
    m.Get("/foo", func() {
  // show something
})

m.Post("/bar", func() {
  // create something
})

Handlers

  • Any kind of callable functions 
    m.Get("/foo", func() {
  // show something
})
  • a handler which is invoked before the routing is called "middleware handler" 
    m.Use(func() {
  // do some middleware stuff
})

m.Get("/foo", func() {
  // show something
})

Features

  • Modular design
  • Service injection
  • Compatible with http package built in Go
